Using the editor

The editor is loaded as flash in HTML page. First download the flash and put it into your web site. it is strongly recommended that load the editor via swfobject. The following line embeds swfobject library from Google CDN.

<script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/swfobject/2.2/swfobject.js"></script>

Next, load the editor via swfobject.embedSWF method. You are able to find enough comments between the lines above if you switch to "JavaScript" radio button. Also you can find details about the initialization options in next section.

After the editor is loaded, you are able to do extra initialization within the javascript callback function (i.e. load the content into the editor). You can control the editor via invoking its methods from javascript. For details, please refer to "Methods" section.

Options

The following options can be set to the flashvar when initialize the editor.

  • parser

    A string value indicating the language parser of the editor.
    It can be any one of the following values :
    asp / aspx / php / jsp / java / csharp / cpp / javascript / sql / xml / css / html.

  • onload

    A string value indicating the javascript callback function. After the editor is initialized, the callback function is called. Extra initialization can be done within the callback function.

  • readOnly [Optional]

    A boolean value indicating if the editor is initialized as readOnly mode.

  • preferredFonts [Optional]

    A string value indicating the preferred font list. The editor detects the installed fonts on the machine, and choose the most preferred font. The default preferred font list is "|Consolas|Courier New|Courier|Arial|Tahoma|". The font list must start and end with the vertical bar charactor "|"

  • fontSize [Optional]

    A integer value indicating the font size in pixel. Default font size is 12.

Methods

The following methods can be invoked via javascript once the editor is initialized.

  • setParser(parseName:String)

    Change the language parser of the editor. parseName can be any one of the following values : asp / aspx / php / jsp / java / csharp / cpp / javascript / sql / xml / css / html

  • setReadOnly(readOnly:Boolean)

    Enable / Disable readOnly mode of the editor.

  • setText(content:String)

    Set the content of the editor.

  • insertText(content:String)

    Replace the current selected text in the editor.
    NOTE : this method will do nothing if the editor is in readOnly mode.

  • getText(content:String)

    Get the content of the editor. Usually this method is used to gather the content and then submit the content to server in your preferred way.
